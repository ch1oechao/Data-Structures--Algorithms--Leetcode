# LinkedList

链表存储有序的元素集合，但不同于数组

链表中的元素在内存中并不是连续放置的

每个元素由一个存储元素本身的节点和一个指向下一个元素的引用(也称指针或链接)组成

### 基本方法

`append(element)`

向列表尾部添加一个新的项

`insert(pos, elem)`

向列表的特定位置插入一个新的项

`remove(elem)`

从列表中移除一项

`indexOf(elem)`

返回元素在列表中的索引 如果没有该元素返回 -1

`removeAt(pos)`

从列表中特定位置移除一项

`isEmpty()`

如果链表中不包含任何元素 返回 true 否则返回 false

`size()`

返回链表包含的元素个数

`print()`

输出元素的值
